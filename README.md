
# task-api

## Descripción
API REST para la gestión de tareas desarrollada con Jakarta EE, Spring Data JPA y Spring MVC.

## Tecnologías
- Java 21
- Spring Data
- PostgreSQL 17
- Docker Compose

## Requisitos
- Java 21 o superior
- Docker y Docker Compose

## Ejecución
1. Para iniciar la aplicación, simplemente ejecuta:
```bash
docker compose up
```
