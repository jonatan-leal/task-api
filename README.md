
# task-api

## Descripción
API REST para la gestión de tareas desarrollada con Spring Boot.

## Tecnologías
- Java 21
- Spring Data
- Spring Security
- JWT
- PostgreSQL 17
- Docker Compose

## Requisitos
- Java 21 o superior
- Docker y Docker Compose

## Ejecución
1. Para iniciar la aplicación, simplemente ejecuta:
```bash
docker compose up
```
